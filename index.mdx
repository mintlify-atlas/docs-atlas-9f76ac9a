## What is Restash JS?

`@restash/restash-js` is a lightweight TypeScript SDK that lets you upload files from the browser directly to the Restash CDN. It handles the full upload lifecycle — getting a presigned URL, streaming the file to S3, and confirming delivery — so you don't have to.

```typescript
import { createRestashUploader } from "@restash/restash-js";

const upload = createRestashUploader({ publicKey: "pk_..." });

const result = await upload(file, {
  path: "uploads/",
  onProgress: ({ percent }) => console.log(`${percent}%`),
});

console.log(result.url); // https://cdn.restash.io/...
```

## Key features

  <Card title="Direct browser uploads" icon="arrow-up-from-bracket">
    Files go straight from the browser to the CDN — no server relay needed.
  </Card>
  <Card title="Progress callbacks" icon="gauge">
    Get byte-level progress events for responsive upload UIs.
  </Card>
  <Card title="Signed upload support" icon="key">
    Require HMAC-SHA256 signatures from your server for secure, authorized uploads.
  </Card>
  <Card title="TypeScript-first" icon="brackets-curly">
    Full type exports for `UploadResult`, `UploadOptions`, and more.
  </Card>
  <Card title="File and Blob support" icon="file">
    Works with both `File` objects from input elements and raw `Blob` data.
  </Card>
  <Card title="Custom paths and metadata" icon="folder">
    Organize uploads into directories and attach arbitrary key-value metadata.
  </Card>
</CardGroup>
