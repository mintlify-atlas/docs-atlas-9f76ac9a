## Upload a file

<Steps>
  <Step title="Create an uploader">
    Call `createRestashUploader` once with your public API key. The returned function is what you'll use to upload files.

    ```typescript
    import { createRestashUploader } from '@restash/restash-js';

    const upload = createRestashUploader({
      publicKey: 'pk_your_public_key',
    });
    ```

    <Note>
      `createRestashUploader` returns an async function. You can create it at module scope and reuse it across multiple uploads.
    </Note>
  </Step>

  <Step title="Get a file from an input element">
    Read the `File` object from a standard HTML file input.

    ```typescript
    const input = document.querySelector<HTMLInputElement>('input[type="file"]');
    const file = input?.files?.[0];

    if (!file) {
      throw new Error('No file selected');
    }
    ```
  </Step>

  <Step title="Upload the file">
    Pass the `File` to the uploader. You can optionally provide a path, a custom name, and a progress callback.

    ```typescript
    const result = await upload(file, {
      path: 'avatars',
      onProgress: ({ percent, loaded, total }) => {
        console.log(`${percent}% uploaded (${loaded} / ${total} bytes)`);
      },
    });
    ```
  </Step>

  <Step title="Use the result">
    The resolved value contains the CDN URL and file metadata.

    ```typescript
    console.log(result.url);         // https://cdn.restash.io/...
    console.log(result.id);          // unique file ID
    console.log(result.key);         // storage key
    console.log(result.name);        // original filename
    console.log(result.size);        // bytes
    console.log(result.contentType); // e.g. "image/png"
    console.log(result.metadata);    // Record<string, string> | null
    ```
  </Step>
</Steps>

## Complete example

```typescript
import { createRestashUploader } from '@restash/restash-js';

const upload = createRestashUploader({
  publicKey: 'pk_your_public_key',
});

async function handleFileInput(input: HTMLInputElement) {
  const file = input.files?.[0];
  if (!file) return;

  const result = await upload(file, {
    path: 'uploads',
    onProgress: ({ percent }) => {
      console.log(`${percent}% complete`);
    },
  });

  console.log('File available at:', result.url);
}
```

## Signed uploads

For production applications, generate a server-side signature before uploading so Restash can verify each request is authorized. Pass your server's signature endpoint URL as `handleSignature` when creating the uploader.

```typescript
const upload = createRestashUploader({
  publicKey: 'pk_your_public_key',
  handleSignature: 'https://your-app.com/api/sign-upload',
});
```

<Tip>
  See the signed uploads guide for how to implement the `handleSignature` endpoint using `generateSig` on your server.
</Tip>
